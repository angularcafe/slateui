CLI

Use the SlateUI CLI to add components to your Angular project.

Installation

Install the SlateUI CLI as a dev dependency in your project:

init

Use the init command to initialize configuration and dependencies for a new project.

The init command installs dependencies, adds the cn util and configures CSS variables for the project.

add

Use the add command to add components and dependencies to your project.

Command Options

init Options

Option Description Default
-t, --template The template to use (angular, angular-monorepo) angular
-b, --base-color The base color to use (neutral, gray, zinc, stone, slate) slate
-y, --yes Skip confirmation prompt true
-f, --force Force overwrite of existing configuration false
-c, --cwd The working directory current directory
-s, --silent Mute output false
--src-dir Use the src directory when creating a new project false
--css-variables Use CSS variables for theming true

add Options

Option Description Default
-y, --yes Skip confirmation prompt false
-o, --overwrite Overwrite existing files false
-c, --cwd The working directory current directory
-a, --all Add all available components false
-p, --path The path to add the component to -
-s, --silent Mute output false
--src-dir Use the src directory when creating a new project false
--css-variables Use CSS variables for theming true
